感動する映画 TOP3

  1. 最強のふたり

    2. 友情や信頼、絆を改めて学ぶことができる、素晴らしい映画です。
    身体に障害を持つ富豪の男性と、スラム街で育つ黒人青年が偶然にも出会い交流を深め、笑いと涙を交えていくストーリーです。
    社会的立場が異なる二人が、障害を持っていても健常者のように扱う青年の行動には心を打たれました。辛い体験をしてきたからこそ、二人の友情は深まったのだと思いました。

    二人の人生の交わりについて気になった方がいればぜひ見てみてください

  2. ワンダー君は太陽

    3. 人の心の強さ、偏見の罪深さを学ぶことができる、素晴らしい映画です。
    障害によって、人とは違う顔を持って生まれてきた少年が困難を立ち向かい、環境を変えていく感動のストーリーです。
    学校に行くことを決めた少年は周りのクラスメートに顔を見られ、軽蔑されてしまいます。しかし、少年は困難に負けず強い心を持って立ち向かいます。生活を行っていく上で学校のクラスメートとの友情がどのようになっていくのか。

    少年の行った行動が気になった方がいればぜひ見てみてください。

  3. 君の膵臓をたべたい

    4. 膵臓に病を持つ女子高校生と男子高校生の青春映画です。
    人生が残り少ないと分かっていても、1日1日を前向きに過ごすことができるのか。

    二人は学校生活をどのように過ごしていったのか興味を持った方がいればぜひ見てみてください。


迫力のある映画 TOP3

  1. ミッション：インポッシブル

    2. トムクルーズの迫力にあるアクションのスパイ映画です。
    今まで6作品が公開され毎度大人気作品として評価されてきました。私は、主人公のCGを使わないアクションが複数あることに驚かされました。誰が味方で誰が敵なのか.....

    トムクルーズのかっこいいアクションが気になった方がいればぜひ見てみてください。

  2. ワイルド・スピード

    3. 車が好きな人にはたまらない、カーアクション映画です。
    今まで10作品が公開され毎度大人気作品として評価されてきました。たくましい主人公の家族を思う笑あり涙ありのストーリーです。

    どんな困難にも立ち向かう主人公のカーアクションが気になった方がいればぜひ見てみてください。

  3. キングスマン

    4. 華麗に任務をこなしていくスパイのアクション映画です。
    今までに2作品が公開され毎度大人気作品として評価されてきました。
    2021年12月には3作品目の公開が決まり今熱い映画です。
    スパイ映画ならではの先の見えない展開と主人公が紳士すぎてキレキレのアクションに見入ってしまいます。

    他の映画にはないスパイ道具などもあり、興味を持った方がいればぜひ見てみてください。

※アクション映画は暴力描写や刺激的なシーンが多いので注意してください。

おすすめの映画シリーズ

Marvel

Marvelとはアメリカ合衆国に本社を置くMrvel comicsと言う出版社が制作しているヒーロー映画のタイトルです。
代表的な作品としてスパイダーマンやアイアンマンなどがあります。一度は聞いたことがあるのではないでしょうか。
一昨年、今年度のマーベル作品は過去一熱いといって過言ではありません。
まず、Marvelは『フェイズ』と呼ばれる話の段落で区切られています。そして、各ヒーロの個人映画が繋がり成り立っています。
過去１暑いといったのは一昨年、全てのフェイズが終了し今年の10月から新たなフェイズが始まることが理由です。

SNSでも話題となったアベンジャーズエンドゲームは興行収入が27.98億ドルとなりました。
今回は中でも私がおすすめする３作品を紹介したいと思います。

Marvelの歴史

marvelは2008年のアイアンマンの映画をきっかけに2019のエンドゲームまで話は続きました。
そして今年から新たなフェイズに進みます。

Marvel 作品

Marvel TOP3

説明しすぎるとネタバレになってしまうため話の大まかな概要だけ述べさせていただきます。

  1. アイアンマン1

    2. 原点の映画です。私がmarvelを好きになったのもこの映画のおかげです。小さい頃に映画館でみて主人公のかっこいい姿を見て大好きになりました。
    主人公のトニースタークはアメリカのトップと言っていいほどの頭の良い学者でした。ある日反政府の団体に拉致されてしまい脱出をするためにパワードスーツというものを作りました。これがアイアンマンの原点でトニーは危機を国民から守ために今まで行なっていた武器製造をやめ、ヒーロとなって活躍することを決めました。

    近未来なアクションが気になった方がいればぜひ見てみてください。

  2. アベンジャーズ／エンドゲーム

    3. マーベル映画の大きな物語の終着点と言っていいこの映画は話の展開が早く常にハラハラさせられる映画でした。私自身ラストシーンには感動しました。
    大きな力を持つ石を利用して地球の存続を壊そうとする敵を倒そうとする物語です。

    この映画はぜひ過去の作品を１度みてからご覧になってください。

  3. ドクター・ストレンジ

    4. 医師で安定した生活を過ごしていたストレンジは、ある日事故に遭い人生から転落してしまいます。
    そこで彼を助けたのが魔術です。厳しい修行を行い人類では直せない後遺症を直しました。
    時間と空間を自由に扱うストレンジが悪と戦う物語です。

    不思議な世界観を味わってみたい方がいればぜひ見てみてください。
